# FedaVest Platform

## Project Overview
FedaVest is a mobile-first, trust and accountability platform that helps food-system SMEs in Nigeria access funding by proving investment readiness and impact. It enables investors to confidently fund, monitor, and verify food-security outcomes aligned with SDG 2 (Zero Hunger).

## Key Features
- **AI Investment Readiness Scoring**: Quantitative assessment of SME financial health.
- **AI Risk & Resilience Detection**: Pattern recognition for supply chain and financial risks.
- **Impact-Aware Matching**: Connecting investors with SMEs based on food security outcomes.
- **Milestone-Based Accountability**: Verified evidence-based funding disbursement.
